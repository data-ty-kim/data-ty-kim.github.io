장학금 데이터 시각화 및 추천 서비스

대학원생을 대상으로 장학금 정보를 시각화하고 맞춤형 추천 기능을 제공하는 웹 기반 서비스를 개발하였습니다. 장학금 및 연구비 데이터를 효과적으로 분석하고 시각화하여, 사용자가 필요한 정보를 직관적으로 탐색할 수 있도록 구성하였습니다. 해당 서비스는 Google Cloud Platform(GCP) 환경에서 서비스를 구축 및 운영하였습니다. UI/UX 디자이너와의 긴밀한 협업을 통해 이용자 중심의 인터페이스를 제공할 수 있었고 그 결과 학생들로부터 높은 만족도를 이끌어냈습니다.

  • 기술: Plotly, Dash, Figma, Dash Bootstrap Component, GCP(Google Cloud Platform), Firestore
  • 성과: 설문조사 만족도 4.21 (5점 만점)
  • 역할: GCP 기반 웹 서비스 구축 및 배포, Firestore DB 설계 및 구축, 시각화 대시보드 구현
