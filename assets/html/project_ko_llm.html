연구 로드맵 및 전공 도서 추천 시스템 (진행 중)

대학원생들의 연구 로드맵과 관련 전공 도서를 추천하는 시스템을 개발 중입니다. 해당 추천시스템은 하이브리드 필터링 방식을 기반으로 합니다. 학생들의 학적 및 연구 데이터와 도서 DB를 통합하여 개인화된 추천을 제공합니다. 추천 모델은 학습된 sLLM을 활용할 예정이며, 지식 그래프 형태로 구성된 Graph DB(neo4j)를 기반으로 관계 기반 탐색과 추론을 수행합니다.

  • Model: 미정(8B 이하의 sLLM 적용 예정)
  • Database: neo4j
  • 역할: 프로젝트 기획, DB 구축, 시스템 아키텍처 설계, 모델 학습
  • 현재 진행상황: Graph DB(neo4j) 구축 완료, 추천 로직 설계 중
