고려대학교 위임전결 금액 변경 효과 분석
Tool:
Pandas, Plotly
Language:
Python, SQL
결과:
위임전결 금액 변경에 기여
(결재문서 건수 36.98% 감소, 시간 18.51% 감소를 추론)
역할:
데이터 분석 및 데이터 시각화
배경:
전자결재 데이터를 분석하여, 위임전결 금액이 변경되는 데 기여하였습니다. 물가상승으로 인해 위임 전결 금액의 상향이 논의되었습니다. 기존에 200만원 이상의 예산집행에 부서장 전결이 필요했다면, 1000만원으로 상향해서 부장의 권한을 높이는 동시에 업무효율성을 추구하자는 것이 핵심이었습니다.
해결방법:
저는 동료들과 Oracle에서 전자결재 데이터를 추출하고 Pandas로 금액대별 예산집행 현황과 변경 효과를 분석하여 예측하였습니다. 이를 위하여 먼저, 업무효율성이란 무엇인지에 대해 다음 두 가지로 정의하였습니다. 첫째, 문서 결재 건수 감소, 둘째, 한 건의 전자문서가 전결에 이르기까지 소요되는 시간 감소입니다. 다음으로 현 상황과 기대효과에 대한 가설을 설정하였습니다. ‘특정 금액에서 부서장 전결이었던 결재 건이 부장전결로 끝난다면, 전체적으로 업무효율성이 증가될 것이다’는 가설을 설정하고 이를 바탕으로 구체적인 분석을 실시했습니다. 각 전자결재 데이터에서 문서번호를 토대로 결재선을 추적하여, 금액을 통해 분류하는 과정을 거쳐, 전결금액이 상향된다면 실제로 ‘결재’를 클릭하는 건수가 36.98% 줄어드는 효과를 확인하였습니다. 한 문서가 전결로 끝나는 데 걸리는 시간 역시 18.51% 감소하였습니다.
